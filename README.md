# Getting Started

## Required Environments
* JDK (>=11)

## 1. git clone
```
git clone https://github.com/tngus3722/GraphQL_RESEARCH.git
cd GraphQL_RESEARCH
```

## 2. build jar
need java (>= 11)
```
./gradlew bootJar 
```

## 3. excute jar
```
java -jar build/libs/*.jar
```

GraphQL GUI web

http://localhost:8080/gui
